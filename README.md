
## 프로젝트 개요

해당 프로젝트는 포시즌즈 펜션의 소개를 위해 만들어진 홈페이지 입니다.
<br/>
빠른 시간 안에 작업 하는 것에 중점을 두었습니다.
<br/>


## 작업 포인트

시원한 분위기의 사이트를 위해서 메인페이지에 사진을 크게 위치시켰고
<br/>
전체적으로 파란색 톤의 사이트를 개발하였습니다.
<br/>
단순한 기능의 소개페이지이므로 깔끔한 디자인에 집중하였습니다.
<br/>
<br/>
